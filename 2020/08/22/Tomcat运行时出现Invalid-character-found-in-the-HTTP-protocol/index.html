这个参数是设置请求头长度的，不过貌似没有解决问题

（3）涉及非法字符的原因
tomcat在7.0.73, 8.0.39, 8.5.7 版本后，对http解析时做了严格限制。RFC3986文档规定，请求url中只允许包含字母（a-zA-Z）、数字（0-9）和 -_.~ 4个特殊字符，以及保留字符! * ’ ( ) ; : @ & = + $ , / ? # [ ])
